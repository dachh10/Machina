# Skill Registry - Machina

## Compact Rules

### General
- Project: Machina (Next.js 14 App Router)
- Stack: TypeScript, Tailwind CSS, Firebase (Auth + Firestore).
- Architecture: App Router structure. Components in `app/components`.
- Roles: "usuario" and "admin". Check `usuarios` collection in Firestore.

### Coding Standards
- Use functional components and React hooks.
- Use `lucide-react` for icons.
- Use `clsx` and `tailwind-merge` for class names.
- Follow Next.js App Router conventions (server vs client components).

### Testing & Quality
- Strict TDD Mode: DISABLED (no test runner found).
- Use `npm run lint` for linting.
- Use `npx tsc --noEmit` for type checking.
